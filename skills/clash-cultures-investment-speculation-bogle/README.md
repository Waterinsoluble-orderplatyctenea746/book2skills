# The Clash of the Cultures: Investment vs. Speculation

A Claude skill distilled from *The Clash of the Cultures: Investment vs. Speculation* by John C. Bogle, founder of Vanguard and creator of the first index fund.

## What It Does

- Evaluates mutual funds, ETFs, and investment products using Bogle's cost framework
- Calculates the long-term compounding impact of investment fees on terminal wealth
- Distinguishes investment (enterprise focus, long-term) from speculation (psychology focus, short-term)
- Identifies agency conflicts in financial advice and fund recommendations
- Explains why index funds mathematically must outperform the average active fund

## When to Use It

Trigger this skill when asked:
- "Is this fund worth investing in?"
- "Should I trade stocks or hold for the long term?"
- "Are my investment costs too high?"
- "Why do most active funds underperform the index?"
- "Should I use a financial advisor?"
- "Is high-frequency trading good or bad for investors?"

## Installation

```bash
npx skills add simbajigege/book2skills/skills/clash-cultures-investment-speculation-bogle
```

1. Download or clone this folder.
2. In Claude.ai, go to **Settings → Skills** and upload the skill folder.
3. The skill will appear in your available skills list.
