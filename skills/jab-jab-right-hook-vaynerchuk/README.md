# Jab Jab Jab Right Hook — Social Media Content Strategy Skill

A Claude skill that applies Gary Vaynerchuk's framework from *Jab, Jab, Jab, Right Hook* to help you build a social media content strategy that gives value first, sells second, and respects the unique language of each platform.

## What It Does

- **Diagnoses your jab-to-right-hook ratio** — identifies whether you're asking too much without giving enough first
- **Evaluates native content quality** — assesses whether your content speaks each platform's language or falls flat as an obvious ad
- **Prescribes platform-specific content** — what does a jab look like on Instagram vs. Twitter vs. Facebook?
- **Builds the 365-day storytelling mindset** — frames social media as a continuous daily practice, not a campaign with a start and end date

## When to Use It

Trigger this skill with phrases like:
- "What should I post on social media?"
- "Why isn't my social media content getting engagement?"
- "How do I sell on social media without feeling salesy?"
- "Should I post the same content on every platform?"
- "How do I build a social media content strategy?"
- "I'm getting followers but no sales"
- "What's the right mix of promotional vs. non-promotional content?"

## Installation

```bash
npx skills add simbajigege/book2skills/skills/jab-jab-right-hook-vaynerchuk
```

1. Download or clone this folder.
2. In Claude.ai, go to Settings → Skills and upload the skill folder.
3. The skill will appear in your available skills list.
